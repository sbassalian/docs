---
title: "Get Creator Payouts"
openapi: "GET /stats/creator_payouts"
---

Retrieves creator payout information including payment amounts, deal dates, and campaign details.

## Request Parameters

| Parameter  | Type    | Description                                                | Required |
| ---------- | ------- | ---------------------------------------------------------- | -------- |
| `page`     | Integer | Page number for pagination. Default is 1.                  | No       |
| `per_page` | Integer | Number of results per page. Default is 40. Maximum is 100. | No       |

## Request Example

```bash
curl -X GET "https://api.growi.io/api/public/v1/stats/creator_payouts?page=1&per_page=20" \
     -H "Authorization: Bearer YOUR_PUBLIC_API_KEY" \
     -H "Content-Type: application/json"
```

## Response Fields

| Field                        | Type    | Description                                           |
| ---------------------------- | ------- | ----------------------------------------------------- |
| `success`                    | Boolean | Indicates whether the request was successful          |
| `data`                       | Array   | Array of creator payout objects                       |
| `data[].id`                  | Integer | Unique identifier for the payout record               |
| `data[].creator_name`        | String  | Display name of the creator                           |
| `data[].creator_avatar_url`  | String  | URL to the creator's avatar image                     |
| `data[].creator_id`          | Integer | Unique identifier for the creator                     |
| `data[].amount`              | String  | Payout amount in formatted currency                   |
| `data[].deal_date`           | String  | Date when the deal was completed (MM/DD/YYYY format)  |
| `data[].since_date`          | String  | Start date of the campaign period (MM/DD/YYYY format) |
| `data[].campaign_category`   | String  | Category of the campaign                              |
| `data[].campaign_name`       | String  | Name of the associated campaign                       |
| `data[].documents`           | String  | Associated documents (null if none)                   |
| `data[].views`               | String  | View status or completion status                      |
| `data[].status`              | String  | Current payout status (e.g., "Pending", "Paid")       |
| `data[].contents_delivered`  | Integer | Number of content pieces delivered by the creator     |
| `data[].contents_required`   | Integer | Number of content pieces required for the campaign    |
| `meta`                       | Object  | Pagination metadata                                   |
| `meta.current_page`          | Integer | Current page number                                   |
| `meta.per_page`              | Integer | Number of results per page                            |
| `meta.row_count`             | Integer | Total number of rows/records                          |
| `meta.page_count`            | Integer | Total number of pages                                 |
| `meta.has_more`              | Boolean | Whether there are more pages available                |
| `additional_data`            | Object  | Summary financial information                         |
| `additional_data.total_paid` | Number  | Total amount already paid to creators                 |
| `additional_data.total_owed` | Number  | Total amount still owed to creators                   |
