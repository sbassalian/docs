---
title: 'Get Top Creators By GMV'
openapi: 'GET /stats/top_creators_by_gmv'
---

Retrieves your top-performing creators ranked by Gross Merchandise Value (GMV) within a specified date range.

## Request Parameters

| Parameter | Type | Description | Required |
| --- | --- | --- | --- |
| `start_date` | String | Start date in MM/DD/YYYY format | Yes |
| `end_date` | String | End date in MM/DD/YYYY format | Yes |
| `limit` | Integer | Limit the number of results. Default is 4. Recommended is 20. Set to 10,000 to get all data. | No |
| `include_discord` | Boolean | Include Discord account ID and username for creators. Default is false. | No |

## Request Example

```bash
curl -X GET "https://api.growi.io/api/public/v1/stats/top_creators_by_gmv?start_date=01/01/2023&end_date=01/31/2023&limit=5&include_discord=true" \
     -H "Authorization: Bearer YOUR_PUBLIC_API_KEY" \
     -H "Content-Type: application/json"
```
