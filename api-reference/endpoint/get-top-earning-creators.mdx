---
title: 'Get Top Earning Creators'
openapi: 'GET /stats/top_earning_creators'
---

Retrieves your top creators ranked by earnings within a specified date range.

## Request Parameters

| Parameter | Type | Description | Required |
| --- | --- | --- | --- |
| `start_date` | String | Start date in MM/DD/YYYY format | Yes |
| `end_date` | String | End date in MM/DD/YYYY format | Yes |
| `limit` | Integer | Limit the number of results. Default is 4. Recommended is 20. Set to 10,000 to get all data. | No |

## Request Example

```bash
curl -X GET "https://api.growi.io/api/public/v1/stats/top_earning_creators?start_date=01/01/2023&end_date=01/31/2023&limit=5" \
     -H "Authorization: Bearer YOUR_PUBLIC_API_KEY" \
     -H "Content-Type: application/json"
```
